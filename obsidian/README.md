# Obsidian vault

This directory is reserved for an Obsidian vault that mirrors the project's
documentation. It is intentionally empty at Phase 1.1.

Future use:

- `docs/adr/*.md` will be linked here.
- Mermaid diagrams in `docs/diagrams/` may be inlined as Dataview blocks.

The vault is git-tracked; plugins and `.obsidian` workspace settings are
ignored via `.gitignore` at the root.
