# @agent-os/shared

Cross-cutting shared types and Zod schemas used by the API and runtime.

## Exports

- `HealthStatusSchema` / `HealthStatus` — health check payload.
- `ErrorEnvelopeSchema` / `ErrorEnvelope` — wire-level error response.
- `PACKAGE_NAME`, `PACKAGE_VERSION` — runtime introspection.

## Status

Phase 1.1 — schemas are skeletons; concrete domain models land in Phase 2.
